All game UI, labels, placeholders, button text, and copy must be in English.

**Why:** User explicitly corrected this after the Value Square was built in German.

**How to apply:** Any new game or UI element — labels like "DEIN WERT", button text like "Vorschläge bekommen", placeholders, section headers — must be in English. The AI responses adapt to the user's input language automatically, but the interface itself is always English.
